## 🧩 **Esercizio 2 — Tupla con un solo elemento**

Crea una tupla che contenga un solo elemento `"Milk"`.

Stampala e verifica il tipo con `type()`.

## 🧩 **Esercizio 3 — Accesso agli elementi**

Data la tupla:

```python
products = ("Bread", "Milk", "Eggs", "Cheese")
```

Stampa:

* il primo elemento
* l’ultimo elemento

## 🧩 **Esercizio 4 — Lunghezza di una tupla**

Data la tupla:

```python
inventory = ("Apples", "Bananas", "Oranges", "Grapes")
```

Stampa il numero di elementi usando `len()`.

## 🧩 **Esercizio 5 — Slicing di una tupla**

Data la tupla:

```python
prices = (1.20, 0.50, 2.00, 1.75, 3.40)
```

Usa lo slicing per ottenere:

* i primi tre prezzi
* gli ultimi due prezzi

## 🧩 **Esercizio 6 — Verifica di appartenenza**

Data la tupla:

```python
categories = ("Fruits", "Vegetables", "Dairy")
```

Scrivi un controllo che stampi:

* `"Category available"` se `"Dairy"` è presente
* `"Category not found"` altrimenti

## 🧩 **Esercizio 7 — Conteggio degli elementi**

Data la tupla:

```python

```

Conta quante volte `"Apple"` compare nella tupla e stampa il risultato.

## 🧩 **Esercizio 8 — Ricerca dell’indice**

Data la tupla:

