# **10 esercizi Python sulle STRINGHE**

Livello: **base → base/intermedio**

---

## 🧩 **Esercizio 1 — Creazione di stringhe**

Crea una variabile `store_name` che contenga il nome di un negozio a tua scelta.

Stampala a schermo.

---

## 🧩 **Esercizio 2 — Lunghezza di una stringa**

Data la stringa:

```python
product = "Chocolate"
```

Stampa il numero di caratteri usando `len()`.

---

## 🧩 **Esercizio 3 — Accesso ai caratteri**

Data la stringa:

```python
word = "Banana"
```

Stampa:

* il primo carattere
* l’ultimo carattere

---

## 🧩 **Esercizio 4 — Slicing semplice**

Data la stringa:

```python
item = "Strawberry"
```

Usa lo slicing per stampare:

* `"Straw"`
* `"berry"`

---

## 🧩 **Esercizio 5 — Concatenazione**

Crea due variabili:

* `item = "Milk"`
* `price = "$2.50"`

Concatena le stringhe per ottenere:

```
Milk costs $2.50
```

---

## 🧩 **Esercizio 6 — Metodo upper() e lower()**

Data la stringa:

```python
category = "Dairy"
```

Stampa:

* la versione tutta maiuscola
* la versione tutta minuscola

---

## 🧩 **Esercizio 7 — Verifica di contenuto**

Data la stringa:

```python
description = "Fresh organic apples"
```

Scrivi un controllo che stampi:

* `"Keyword found"` se `"apples"` è presente
* `"Keyword not found"` altrimenti

---

## 🧩 **Esercizio 8 — Metodo replace()**

Data la stringa:

```python
message = "Low stock warning"
```

Sostituisci `"Low"` con `"Critical"` e stampa il risultato.

---

## 🧩 **Esercizio 9 — Metodo split()**

Data la stringa:

```python
items = "milk,bread,eggs,butter"
```

Usa `split()` per ottenere una lista di elementi.

Stampa la lista.

---

## 🧩 **Esercizio 10 — Formattazione dell’output**

Usa le variabili:

```python
product = "Apple"
price = 1.20
```

Stampa la frase:

```
The price of Apple is $1.20
```

utilizzando una **f-string**.
