# **10 esercizi Python sulle LISTE**

Livello: **base → base/intermedio**

---

## 🧩 **Esercizio 1 — Creazione di una lista**

Crea una lista chiamata `fruits` che contenga:

```
"apple", "banana", "cherry"
```

Stampala a schermo.

---

## 🧩 **Esercizio 2 — Accesso tramite indice**

Data la lista:

```python
items = ["milk", "bread", "eggs", "cheese"]
```

Stampa:

* il primo elemento
* l’ultimo elemento

---

## 🧩 **Esercizio 3 — Lunghezza della lista**

Data la lista:

```python
inventory = ["apples", "bananas", "oranges"]
```

Stampa il numero di elementi presenti usando `len()`.

---

## 🧩 **Esercizio 4 — Modifica di un elemento**

Data la lista:

```python
prices = [1.20, 0.99, 2.50]
```

Modifica il secondo prezzo in `1.10` e stampa la lista aggiornata.

---

## 🧩 **Esercizio 5 — Aggiungere elementi**

Crea una lista vuota chiamata `shopping_cart`.

Aggiungi:

* `"milk"`
* `"eggs"`
* `"bread"`

Usando `append()`, poi stampa la lista.

---

## 🧩 **Esercizio 6 — Rimuovere un elemento**

Data la lista:

```python
products = ["apples", "bananas", "carrots", "potatoes"]
```

Rimuovi `"carrots"` e stampa la lista aggiornata.

---

## 🧩 **Esercizio 7 — Verifica di appartenenza**

Data la lista:

```python
available_items = ["milk", "eggs", "butter"]
```

Scrivi un controllo che stampi:

* `"Item available"` se `"eggs"` è nella lista
* `"Item not available"` altrimenti

---

## 🧩 **Esercizio 8 — Ordinamento**

Data la lista:

```python
numbers = [5, 2, 9, 1, 7]
```

Ordina la lista in ordine crescente e stampala.

---

## 🧩 **Esercizio 9 — Iterare una lista**

Data la lista:

```python
fruits = ["apple", "banana", "cherry"]
```

Usa un ciclo `for` per stampare ogni elemento su una riga diversa.

---

## 🧩 **Esercizio 10 — Lista mista**

Crea una lista chiamata `product_info` che contenga:

* nome prodotto (stringa)
* quantità (intero)
* disponibile (boolean)
* prezzo (float)

Stampa ogni elemento usando gli indici.
