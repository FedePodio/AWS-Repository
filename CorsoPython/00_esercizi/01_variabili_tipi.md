# **10 esercizi Python** pensati per **competenze base**

---

## 🧩 **Esercizio 1 — Variabili e stampa**

Crea due variabili:

* `product`
* `price`

Assegna:

```text
product = "Apple"
price = 1.20
```

Stampa:

```
The product Apple costs 1.2 dollars
```

---

## 🧩 **Esercizio 2 — Tipi di dato**

Crea le seguenti variabili:

* `item_name` (stringa)
* `quantity` (intero)
* `available` (booleano)

Stampa il valore **e il tipo** di ciascuna variabile usando `type()`.

---

## 🧩 **Esercizio 3 — Input dell’utente (stringa)**

Chiedi all’utente il nome di un prodotto usando `input()`
Salvalo in una variabile `product_name`.

Stampa:

```
You selected: <product_name>
```

---

## 🧩 **Esercizio 4 — Input numerico e conversione**

Chiedi all’utente:

* il prezzo di un prodotto (stringa)
* la quantità acquistata (stringa)

Converti:

* il prezzo in `float`
* la quantità in `int`

Stampa il totale:

```
Total cost: <total>
```

---

## 🧩 **Esercizio 5 — Operazioni matematiche base**

Crea due variabili:

```python
price = 2.50
quantity = 4
```

Calcola:

* `total_cost`

Stampa:

```
Total cost is 10.0
```

```
Customer name: Mario Rossi
```

## 🧩 **Esercizio 7 — Output formattato (f-string)**

Crea:

```python

```

```
The price of Milk is $1.99
```

Stampa:

```
Is the product available? <True/False>
```
