# **10 esercizi Python sui SET**

Livello: **base → base/intermedio**

---

## 🧩 **Esercizio 1 — Creazione di un set**

Crea un set chiamato `fruits` che contenga:

```
"apple", "banana", "orange", "apple"
```

Stampalo e osserva il risultato.

---

## 🧩 **Esercizio 2 — Verifica di appartenenza**

Dato il set:

```python
colors = {"red", "green", "blue"}
```

Verifica se `"green"` è presente nel set e stampa il risultato.

---

## 🧩 **Esercizio 3 — Aggiungere un elemento**

Dato il set:

```python
items = {"bread", "milk"}
```

Aggiungi `"eggs"` al set e stampalo.

---

## 🧩 **Esercizio 4 — Rimuovere un elemento**

Dato il set:

```python
tools = {"hammer", "screwdriver", "wrench"}
```

Rimuovi `"screwdriver"` dal set e stampa il risultato.

---

## 🧩 **Esercizio 5 — Lunghezza di un set**

Dato il set:

```python
numbers = {1, 2, 3, 4, 5}
```

Stampa quanti elementi contiene.

---

## 🧩 **Esercizio 6 — Eliminare duplicati da una lista**

Data la lista:

```python
names = ["Alice", "Bob", "Alice", "Charlie", "Bob"]
```

Usa un set per ottenere una collezione senza duplicati
e stampala.

---

## 🧩 **Esercizio 7 — Unione di set**

Dati i set:

```python
set_a = {1, 2, 3}
set_b = {3, 4, 5}
```

Crea un nuovo set che contenga tutti gli elementi di entrambi.

---

## 🧩 **Esercizio 8 — Intersezione di set**

Usando gli stessi set dell’esercizio precedente,
trova gli elementi **comuni** a `set_a` e `set_b`.

---

## 🧩 **Esercizio 9 — Differenza di set**

Sempre con:

```python
set_a = {1, 2, 3}
set_b = {3, 4, 5}
```

Trova:

* gli elementi presenti in `set_a` ma non in `set_b`

---

## 🧩 **Esercizio 10 — Confronto tra set**

Dati i set:

```python
class1 = {"Alice", "Bob", "Charlie"}
class2 = {"Charlie", "Diana"}
```

Stampa:

* gli studenti presenti in **entrambi** i corsi
* gli studenti presenti **solo nel primo corso**
