# **10 esercizi Python sul Control Flow**, pensati per consolidare l’uso di

* `if`
* `elif`
* `else`
* confronti logici
* condizioni multiple (`and`, `or`)

Livello: **base → intermedio iniziale**

---

## 🧩 **Esercizio 1 — Controllo maggiore/minore**

Crea una variabile `number`.

* Se `number` è maggiore di 10, stampa `"Greater than 10"`
* Altrimenti stampa `"10 or less"`

---

## 🧩 **Esercizio 2 — Disponibilità prodotto**

Crea:

```python
stock = 8
```

* Se lo stock è minore di 10 → `"Low stock"`
* Altrimenti → `"Stock available"`

---

## 🧩 **Esercizio 3 — Idoneità allo sconto**

Crea:

```python
total_cost = 35.00
```

* Se `total_cost >= 20` stampa `"Discount applied"`
* Altrimenti `"No discount"`

---

## 🧩 **Esercizio 4 — Valutazione del voto**

Crea:

```python
score = 82
```

Usa `if / elif / else`:

* ≥ 90 → `"Excellent"`
* ≥ 70 → `"Good"`
* Altrimenti → `"Needs improvement"`

---

## 🧩 **Esercizio 5 — Giorno della settimana**

Crea:

```python
day = "Sunday"
```

* Se `day` è `"Saturday"` o `"Sunday"` → `"Weekend"`
* Altrimenti → `"Weekday"`

---

## 🧩 **Esercizio 6 — Controllo accesso**

Crea:

```python
age = 17
```

* Se `age >= 18` → `"Access granted"`
* Altrimenti → `"Access denied"`

---

## 🧩 **Esercizio 7 — Sconto per categoria**

Crea:

```python
category = "Fruits"
```

* `"Fruits"` → `"10% discount"`
* `"Vegetables"` → `"15% discount"`
* Altrimenti → `"No discount"`

---

## 🧩 **Esercizio 8 — Controllo stock avanzato**

Crea:

```python
stock = 120
```

* Se `stock < 20` → `"Restock needed"`
* Se `stock > 100` → `"Apply discount"`
* Altrimenti → `"Stock level OK"`

---

## 🧩 **Esercizio 9 — Condizioni multiple**

Crea:

```python
product = "Milk"
day = "Monday"
```

* Se il prodotto è `"Milk"` **e** il giorno è `"Monday"` → `"Special offer"`
* Altrimenti → `"Regular price"`

---

## 🧩 **Esercizio 10 — Controllo annidato**

Crea:

```python
product_type = "Perishable"
days_in_store = 5
```

* Se `product_type` è `"Perishable"`:

  * Se `days_in_store > 3` → `"Apply discount"`
  * Altrimenti → `"Product is fresh"`
* Altrimenti → `"Non-perishable product"`
